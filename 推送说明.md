## 🚀 推送到GitHub

### 方法1：推荐 - 使用GitHub CLI

```bash
# 1. 安装GitHub CLI（如果未安装）
brew install gh

# 2. 登录GitHub
gh auth login

# 3. 进入测试数据目录
cd /Users/tong/Code/AI/ppmenu-assets-test

# 4. 推送到test分支
git push -u origin test
```

### 方法2：使用SSH密钥

```bash
# 1. 进入测试数据目录
cd /Users/tong/Code/AI/ppmenu-assets-test

# 2. 检查远程地址
git remote -v

# 3. 如果是HTTPS，改为SSH
git remote set-url origin git@github.com:tKyle-Libra/ppmenu_assets.git

# 4. 推送
git push -u origin test
```

### 方法3：使用Personal Access Token

```bash
cd /Users/tong/Code/AI/ppmenu-assets-test
git push -u origin test
# 输入GitHub用户名和Token（不是密码）
```
